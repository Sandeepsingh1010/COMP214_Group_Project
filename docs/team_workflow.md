# Team Workflow

## Roles
- Member 1: tasks 1-2
- Member 2: tasks 3-4
- Member 3: tasks 5-6
- Member 4: tasks 7-9

Adjust however your group prefers.

## Rules
- One task per branch
- One task per SQL file
- Put test code in `sql/03_tests/`
- Keep the final combined file clean and runnable
- Review before merge

## Recommended PR checklist
- Compiles in Oracle SQL Developer
- Has exception handling, including `WHEN OTHERS`
- Has sample invocation / test block
- Comments explain logic
